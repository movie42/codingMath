캔버스 생성

  • 캔버스 요소에는 width와 height 두 속성만 있다. 선택사항이고 DOM프로퍼티로 설정이 가능하다.
  • width와 height 속성을 지정하지 않으면 width 300, height 150이 기본으로 지정된다.
  • CSS 크기 지정이 초기 캔버스의 비율을 고려하지 않으면 이미지가 왜곡되어 나타난다.

대체 콘텐츠

current stock price: $3.15 +0.15

canvas를 지원하지 않는 브라우저를 위해 사용함. canvas안에 대체 내용을 넣으면 된다.

랜더링 컨텍스트

  • 2D, 3D가 가능하다.
  • getContext()를 사용해 랜더링 컨텍스트와 그리기 함수를 사용할 수 있다.

예

캔버스 가져오기

let canvas = document.getElementById('canvas')

컨택스트 지정

let ctx = canvas.getContext('2d')
